# Görseller Klasörü

Bu klasöre şunları koy:

- `favicon.png` — Tarayıcı sekmesi ikonu (32x32 veya 64x64 px)
- `logo.png` — Site logosu (önerilen: 280x72px, şeffaf PNG)
- `default-sponsor.jpg` — Sponsor görseli bulunamazsa gösterilecek varsayılan

## Sponsor Logoları
Sponsor logoları URL olarak admin panelinden girilir.
Yerel logo kullanmak istersen buraya yükle ve URL olarak `/assets/images/sponsor-adi.png` gir.

## Önerilen Boyutlar
- Sponsor logo: kare, min 300x300px, PNG (şeffaf arka plan en iyi görünür)
- Banner/GIF: 1200x80px (mobil) / 1200x180px (masaüstü)
- Nav logo: 280x72px (geniş) veya kare 72x72px
